# 为啥开源这个框架?

## 起因

去年的时候, 心血来潮. 我决定和不不开一个设计补环境框架的课. 内容是关于修改node源码+js框架相结合的补环境框架(主要是优化+过一些棘手的检测点)

然后就基于上一版自己写的框架基础上, 重构优化了一遍, 就是现在开源的这版.
